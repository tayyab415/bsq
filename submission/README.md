# Submission zip (not committed as a whole)

Build `TeamName.zip` locally with:

```text
TeamName.zip
  github_link.txt
  presentation_video.mp4
  executive_summary.pdf
  prfaq.pdf          # optional
```

## github_link.txt

Copy `github_link.txt.example` → `github_link.txt` and set your public (or private + MoellerO) URL.

## Videos

- Source: `shooting-videos/outputs/` or screen recording of `visualizer/`
- Constraints: **max 3 minutes**, resolution **below 720p**

## Slides

Build `executive_summary.pdf` externally; link to this repo for reproduction steps.
